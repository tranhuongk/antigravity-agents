---
description: Reproduce, isolate, fix, and verify a failure in any project.
---

When the user runs `/codex-debug <failure or request>`, use the `codex-debug-fix` skill:

1. Capture the exact failure.
2. Reproduce with the narrowest command.
3. Search and read the relevant code.
4. Fix the root cause with a minimal diff.
5. Add a regression test when feasible.
6. Rerun the failing command.
7. Report cause, fix, verification, and residual risk.
