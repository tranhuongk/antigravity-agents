When the user runs `/codex-docs <request>`, use the `codex-docs-research` skill:

1. Read project source-of-truth docs and code first.
2. Use official or primary sources for external/current facts.
3. Keep commands and examples copy-pasteable.
4. Update docs close to the behavior they describe.
5. Verify commands and links when feasible.
6. Cite external sources when used.
